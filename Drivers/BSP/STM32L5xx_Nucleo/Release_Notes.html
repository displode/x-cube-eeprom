  • USB Power Delivery

Contents

First official release of board drivers for NUCLEO-L552ZE-Q board (MB1361-C02) in line with STM32Cube BSP drivers development guidelines (UM2298 - revision 2)

Dependencies

This software release is compatible with:

  • BSP Common v6.0.0 or above
